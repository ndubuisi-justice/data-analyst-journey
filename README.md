## Data Sources

This repository uses datasets from a variety of sources, including:

- Public datasets (such as Kaggle)
- Sample datasets created for practice
- Online learning platforms

Where datasets are subject to copyright or licensing restrictions (for example, course materials provided through online learning platforms), they are **not included** in this repository. Instead, the source of the dataset is referenced in the corresponding exercise documentation.
